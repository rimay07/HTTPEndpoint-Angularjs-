1. Application Dependencies:
- Install latest Nodejs from https://nodejs.org/en/
- Once installed update npm by opening terminal and typing the command 'npm update'
2. Download the application from the GIT repository
- Open Terminal window
- Create new directory with command "mkdir remotestaff"
- type git clone --depth=16 https://github.com/davidplumridge/example_node_angular_task_87512.git
- enter appropriate username and password
4. Opening application:
- go to application root folder (<drive>:remotestaff)
- type npm start
- Once server is started
5. open browser and type localhost:8080/index.html
